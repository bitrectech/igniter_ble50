## 设计初衷

模型火箭的点火时,为了保障安全,通常要保证和模型火箭有足够的安全距离,并且有线部署比较繁琐。离线为最佳的选择。为什么不选择wifi？使用wifi的情境下，手机作为客户端连接点火装置,这样会导致手机的网络受限；而手机作为热点，点火装置端配网操作比较繁琐；为什么选择蓝牙？蓝牙低功耗版本5支持300m以内的连接，连接比较简易。
